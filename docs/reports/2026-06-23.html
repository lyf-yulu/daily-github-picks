TokDiet ⭐ 69

AI编码代理的本地计费与省钱代理

TokDiet 是一款部署在本地的流式反向代理，位于 AI 编码代理（Claude Code、Cursor、Codex）与各大模型 API（Anthropic、OpenAI、Gemini、MiniMax）之间。它的核心定位是为按 Token 计费的开发场景提供透明的成本监控与优化：一方面以 ccusage 风格逐请求计量 Token 消耗与美元成本，并通过本地实时仪表盘呈现使用趋势；另一方面通过上下文压缩（context compaction）技术裁剪冗余的对话历史与无效上下文，从源头降低 API 调用支出。为了确保压缩不会牺牲输出质量，项目还内置了 shadow-eval 影子评估机制，对压缩前后的响应进行对照验证，量化质量影响。整体方案兼顾省钱、可观测和可信三大诉求，适合重度使用 AI 编程助手的个人与团队在本地侧透明管理多家模型供应商的支出。
