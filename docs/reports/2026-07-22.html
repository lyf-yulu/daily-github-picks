Agent Council 智能体议会 ⭐ 91

多 AI CLI 协作汇聚多元观点

Node.jsBash ShellYAMLClaude Code SkillCodex CLIGemini CLI
Agent Council 是一款面向 Claude Code 的多智能体协作插件（Skill），通过调用用户本地已安装的多种 AI CLI（如 Claude Code、Codex CLI、Gemini CLI 等）来收集多方观点，并由可配置的 Chairman（主席）综合出最终结论。项目灵感来自 Karpathy 的 LLM Council，但关键差异在于复用用户已有的 CLI 订阅，无需额外的 API 调用费用。整体流程分为三个阶段：所有成员并行接收同一问题并独立作答、汇总并格式化展示各方回答、由宿主 Agent 或指定 CLI 作为主席进行综合归纳。支持通过 npx 一键安装或以 Claude Code Plugin 形式接入，会根据机器上已存在的 CLI 自动生成 council.config.yaml 配置。提供 start/status/results/clean 等作业流命令，以及一次性执行模式，并针对宿主工具 UI 返回 wait JSON 以驱动原生进度视图，适合需要多视角评审、技术选型讨论的开发场景。

AI GatewayMCP (Model Context Protocol)A2A (Agent-to-Agent)PWADesktop AppToken 压缩 (RTK/Caveman)
这是一个基于 MIT 协议开源的免费 AI 网关项目，旨在通过统一的 API 端点聚合来自 268+ 家提供商的 500+ 个大语言模型，其中包含 50+ 个可免费调用的服务，覆盖 Kimi、Claude、GPT、OpenAI、Gemini、GLM、DeepSeek、MiniMax 等主流模型。项目的核心定位是让开发者永不中断编码工作——通过配额感知的自动故障转移机制，当某个提供商达到限额或不可用时，网关会自动切换到备用通道，保证 AI 编码助手持续可用。在生态兼容性方面，该网关无缝对接 Claude Code、Codex、Cursor、OpenCode、Cline、Copilot 等主流 AI 编程工具，开发者无需修改客户端即可享受统一路由能力。技术亮点上，项目集成了 RTK 与 Caveman 双重压缩算法，可将 token 消耗降低 15% 到 95%，显著节省调用成本；同时支持 MCP（Model Context Protocol）与 A2A（Agent-to-Agent）协议，具备现代 Agent 生态的互操作能力。交付形态上提供桌面客户端与 PWA 渐进式 Web 应用两种方式，兼顾本地体验与跨平台便利。项目由 500+ 位社区贡献者共同构建，展现出较为活跃的开源协作氛围。
在 AI 编程助手日益成为核心生产力工具的当下，模型限额、账号封禁、成本高企是开发者面临的三大痛点。该项目以免费+MIT 协议+自动 fallback 三重组合直击这些痛点，同时通过 token 压缩带来的成本节约与主流工具的即插即用能力，形成了极高的实用价值。500+ 贡献者的社区规模也说明其获得了广泛认可，值得关注其架构演进与生态整合思路。

LLMVault ⭐ 220

OWASP LLM Top 10 靶场训练平台

Python 3.10+FlaskGunicornDocker / Docker ComposeFernet (cryptography)SVG/PNG 前端渲染JSON 文件持久化
LLMVault 是一个专为 AI 安全教育设计的、故意存在漏洞的 CTF 风格训练靶场，对标 WebGoat 与 KubeGoat，但聚焦于大语言模型应用安全。项目围绕 OWASP LLM Top 10 (2025) 构建了 25 个分层实验：核心层 10 个（每个 OWASP 类别对应一个基础实验，覆盖提示注入、敏感信息泄露、供应链、数据投毒、输出处理、过度代理、系统提示泄露、向量与嵌入、错误信息、无界消耗）；进阶层 10 个多轮对话挑战（角色扮演越狱、碎片重建、工具链攻击、跨租户 RAG 泄露、查询式模型抽取等）；专家层 5 个模拟真实世界漏洞的加密挑战，需通过完成前两层并向作者索取 Expert Access Key 后使用 Fernet 解密。每个实验都配有攻击与防御两面的教学内容，解题后会展示「Learn — the fix」防御课程面板。技术实现上采用 Flask 后端，脚本化确定性的假 LLM 响应保证离线可复现，无需真实 API Key；进度以 data/progress.json 持久化，避免引入数据库；支持 Docker Compose 一键部署，通过 gunicorn 单 worker 保障内存计分板一致性。玩家体验方面提供首次运行姓名锁定、引导教程、响应式布局、可分享的通关卡片（PNG/SVG 服务端渲染）与 LinkedIn/X 分享链。
在生成式 AI 安全人才严重短缺的当下，LLMVault 把抽象的 OWASP LLM Top 10 规范落成可动手复现的靶场，攻防并重且完全离线可跑，极大降低了 AI Red Team 培训门槛。它把 CTF 关卡机制、加密挑战解锁、社交分享通关卡等游戏化设计融入安全教育，是继 WebGoat 之后 AI 时代少见的成体系训练平台，对企业 AI 安全培训与个人渗透测试进阶均有直接参考价值。

mentor 会话洞察技能 ⭐ 56

为 AI 编程代理生成个人工作复盘报告

PythonuvSKILL.md 技能规范HTML/SVGClaude CodeOpenAI Codexskills.sh (npx)
mentor 是一款面向 AI 编程代理（Claude Code、Codex 等支持 SKILL.md 格式的代理）的会话洞察技能。它在本地读取 ~/.claude 下的 Claude Code 会话记录和 ~/.codex 下的 Codex rollout 文件，聚合近 30 天（可通过 --days 调整）的消息数、会话数、工具调用、编辑量、涉及文件与 token 消耗等指标，并交由代理生成八段式分析，最终输出一份自包含的 report.html 报告。报告涵盖工作主题聚类、工具与语言使用比例、亮点成果、摩擦点及可执行修复、值得尝试的能力、CLAUDE.md 追加建议、新工作流与趋势预测，特别强调对每个摩擦点给出可复用的 CLAUDE.md 配置行、Hook 或习惯调整。整个流程完全本地运行，转录内容不上传，保证隐私。技能由三段脚本驱动：collect.py 解析会话产出统计包，代理基于摘要撰写分析，render_report.py 将统计与分析合并渲染为带内嵌 SVG 图标、计数动效和动画图表的 HTML。可通过 skills.sh 一键安装，也支持手动 git clone 到技能目录。
