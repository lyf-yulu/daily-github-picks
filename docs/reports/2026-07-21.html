Node.jsBashYAMLClaude Code SkillsCodex CLIGemini CLI
Agent Council 是一个面向 Claude Code 的多智能体协作技能插件，灵感来自 Karpathy 的 LLM Council，但改用本地已安装的 AI CLI（Claude Code、Codex CLI、Gemini CLI 等）来收集多方意见，因此无需额外支付 API 费用，只需依赖各家 CLI 的订阅计划。项目采用三阶段流程：先并行向所有成员 CLI 发送同一问题收集初始观点，再统一汇总展示，最后由默认的宿主 Agent 或可配置的 Chairman CLI 综合出最终建议。用户可通过 npx 一键安装到 .claude/ 或 .codex/ 目录，也支持 Claude Code 插件市场安装，并通过 council.config.yaml 灵活配置成员、命令、表情与颜色。脚本 council.sh 提供 start、status、results、wait、clean 等子命令，支持后台作业、进度轮询、检查表视图以及一键 one-shot 模式，方便在宿主终端 UI 中集成原生 plan/todo 展示。

LLMVault ⭐ 213

Python 3.10+FlaskGunicornDockerDocker ComposeFernet (cryptography)SVG/HTML5 Canvas
LLMVault 是一个刻意设计为脆弱的 CTF 风格训练平台，专注于 OWASP LLM Applications Top 10 (2025) 的攻防实战教学，可视作 AI 领域的 WebGoat 或 KubeGoat。项目共提供 25 个实验，分为三个递进解锁的层级：核心层（Core）覆盖 OWASP LLM Top 10 每个类别一个实验，涉及提示注入、敏感信息泄露、供应链、数据投毒、输出处理不当、过度授权、系统提示泄漏、向量检索越权、错误信息、无界消耗；进阶层（Advanced）提供 10 个多轮对话挑战，涵盖角色扮演越狱、片段重构、在线学习投毒、二阶注入、Agent 工具链 SSRF、跨租户 RAG 内存泄漏、基于查询的模型抽取等真实攻击手法；专家层（Expert）5 个实验则以 AES-Fernet 加密方式内置于 challenges/expert.enc，需玩家完成前两层并向作者索取密钥后方可解密。项目通过确定性脚本化的假 LLM 实现全离线运行，无需 API Key，flag 可复现。技术上基于 Python 3.10+ 与 Flask，支持 Docker Compose 部署，使用 gunicorn 单 worker 保证内存计分板一致性，进度以 data/progress.json 文件持久化，无需数据库。每个实验解锁后展示防御课程面板，并支持 SVG/PNG 完成证书导出与社交平台分享。
在 AI 安全教育资源仍以文档和 PPT 为主的当下，LLMVault 是少见的、系统化对齐 OWASP LLM Top 10 (2025) 的可交互靶场。它把提示注入、Agent 工具链滥用、RAG 越权、模型抽取等前沿攻击类以多轮对话形式落地为可复现实验，且完全离线运行，规避了 API 成本与不稳定问题，非常适合个人自学、企业红队培训、CTF 命题以及高校 AI 安全课程使用。

Equity Research Skill 是一个面向 Claude Code、Claude Desktop（Cowork）、Codex CLI 等 AI Agent 工具的 Agent Skill，用纯 Markdown 指令把「研究一下某只股票」这类自然语言请求转化为一份事实可溯、时间戳齐全、结论明确的中文机构级投研报告。项目覆盖美股、港股、A股，并针对 A/H 双重上市标的内置溢价对比模块，分市场给出结论。工作流分为五步：确认标的、探测可用数据源并并行采集（行情、Morningstar、财报与卖方观点，工具缺失时自动降级并显式标注）、数据对账与时间戳、按九章模板撰写、估值交叉验证并交付文件。报告严格遵循九章结构：一页速览、业务详情、竞争与护城河、管理层与治理、财务分析、多方法估值、分析师观点汇总、新闻与催化剂、投资结论。估值纪律要求至少三种方法交叉验证——反向 DCF 证伪现价、三情景概率加权 DCF、三要素/EPV（含 EPV÷净资产护城河验证与 franchise 成长）以及相对估值或 SOTP，所有计算由 scripts/dcf.py 执行、假设以 JSON 留档，禁止心算。买卖动作通过预注册标定规则映射输出，确保同一组数字不会给出两种结论。依赖上仅需联网搜索与 Python 3 标准库，IBKR MCP、Morningstar MCP、docx 技能均为可选增强。
在 Agent Skill 生态中，多数示例停留在轻量写作或代码辅助，本项目把「专业金融研究」这类高知识密度、高纪律要求的场景做成了可复现的工作流：既约束了 LLM 最擅长「幻觉数字」的估值环节（强制脚本计算 + 预注册结论映射），又通过降级表把数据源不确定性显式化，是把 AI Agent 从「玩具」推向「专业工具」的一个值得研究的范式样本。
