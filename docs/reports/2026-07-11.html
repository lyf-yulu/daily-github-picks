SAIL Skill 是由 Pillar Security 开源的 Agent 技能包，将其自研的 SAIL V2（Secure AI Lifecycle）框架封装为可即插即用的 SKILL.md 格式技能。项目核心资产是一份覆盖 AI/Agent 全生命周期 7 个阶段、共 91 条风险的完整目录，每条风险都附带缓解措施以及与 ISO/IEC 42001、EU AI Act、OWASP LLM/Agentic、DASF、AIUC-1 等主流合规框架的映射关系。技能会在用户涉及 AI 安全话题时自动触发，让 Agent 基于真实目录条目（真实的 SAIL ID 与映射）回答问题，避免臆造控制项。典型能力包括：对目标 Agent 或系统做完整的 91 项风险差距评估、生成 7 阶段安全路线图与成熟度画像、按框架生成合规检查清单、优先级排序、跨团队对齐产物，以及生成面向安全供应商或平台供应商的 RFP 问卷。仓库同时是一个 Claude Code 插件市场，在 Claude Code 中额外提供 /sail、/sail:assess、/sail:roadmap、/sail:comply、/sail:rfp 五个斜杠命令。安装方式极为广泛，覆盖 Claude Code、Claude.ai/Desktop/Cowork、OpenAI Codex CLI、Google Antigravity、ChatGPT、pi、opencode 等所有兼容 SKILL.md 的 Agent 环境，并专门给 AI Agent 自助安装写了指引。
在 Agent Skills 生态刚刚成型的当下，这是少见的把成熟安全框架产品化为跨 Agent Skill 的实践——一份资产同时服务 Claude Code、ChatGPT、Codex、Antigravity 等主流 Harness，用真实目录条目消除 LLM 在合规输出时的幻觉。对于关注 AI/Agent 安全治理、企业合规落地或想学习如何把领域知识封装为 Skill 的团队，都有直接参考价值。

Claude Antigravity Agents ⭐ 100

让 Claude Code 委派任务给 Antigravity 子代理

Claude Code Skill (Markdown SKILL.md)Google Antigravity CLI (agy)Gemini 模型Git worktreeBash / PowerShell 安装脚本
Claude Antigravity Agents 是一个 Claude Code 技能包（Skill），通过一个名为 antigravity-agents 的 SKILL.md 指令文件，把 Claude Code 从单一执行者升级为编排者（Orchestrator）。安装后，Claude Code 可将耗时的编码、代码审查、架构分析、依赖审计和研究调研等重任务，以非交互方式（agy -p）委派给 Google Antigravity CLI（agy）驱动的子代理，后者由 Gemini 等模型驱动，可在后台并行运行。项目核心价值在于三点：节省 Claude 上下文和订阅 Token、引入第二模型获得独立第二意见、以及多任务并行提升吞吐。安全模型强调隔离与验证：只读任务在沙箱中运行，写文件任务在独立的 git worktree 或分离仓库中执行，绝不触碰 Claude 当前正在编辑的文件；子代理返回结果后，Claude 会通过 git diff、类型检查和构建流程进行验证，才会将其合并。支持 macOS、Linux 与 Windows，采用 MIT 许可证。

JSON SchemaYAMLPythonpytestGitHub ActionsMarkdownGitHub Pages
Cognitive-Core-Skills 是一个通用、行业中立的认知核心技能分类体系（Taxonomy），旨在为 LLM、SLM、AI Agent 以及世界模型（World Model）提供一套统一的能力描述框架。项目将智能体的心智操作能力划分为八大类：感知（Perception）、记忆（Memory）、推理（Reasoning）、规划（Planning）、行动（Action）、验证（Verification）、学习（Learning）与治理（Governance），共覆盖 13 个领域、159 张技能卡片。仓库同时提供机器可读的 JSON/YAML 数据、JSON Schema 校验规则、Markdown 技能卡片、评估基准（rubric-task-traces）以及配套的 pytest 测试和 GitHub Actions CI 流水线。它还额外扩展了文本类技能、人类动作类技能、LLM Wiki 持久化知识操作、以及 OKF 风格的知识编目操作，并引入了 Cognitive Debt Index（认知债务指数）与评估评分表。项目定位并非再造一个模型，而是为 Agent 工程师提供一份可直接引用、可 CI 校验、可与自有栈集成的能力字典，帮助团队区分 LLM 与 Agent 的边界、明确验证与治理关卡，并为多智能体系统的能力盘点、缺口分析与合规审查提供统一话语体系。
在 Agent 生态从 Demo 走向生产的当口，行业普遍缺乏统一的能力描述与验证标准。该项目以 MIT 协议开源了一份工业中立、可机读、带 CI 保障的认知能力分类体系，填补了 Agent 能力盘点、缺口评估与治理审计的空白，对构建可评测、可对齐的智能体架构具有直接参考价值。

Ditto ⭐ 95

把AI聊天记录挖成你的智能体画像

Python (stdlib only)Markdown (提示词与产物)Claude Code / Codex / Cursor SkillsJSONL 会话日志解析
Ditto 是一个本地化的个人画像挖掘工具，将用户在 Claude Code、Codex、Cursor、Gemini 等 AI 编程助手上积累的会话日志转化为一份 you.md 智能体配置文件。它的核心理念是：真正记录你思考方式的不是显式写下的 CLAUDE.md 或规则文件，而是你在日常工作中与 AI 交互时留下的原始对话——你如何拒绝、如何定义完成、何时要求验证、真实的表达风格等。工具首先从本地日志中仅提取用户输入的文字（剥离所有工具输出与粘贴内容），在写出前完成密钥与 PII 脱敏，再将语料切分为约 70K token 的多个 chunk。用户在 AI Agent 中粘贴内置的 MINING_PROMPT，让 20 个 Agent 并行分析这些 chunk，最终按共识数排序合并出 you.md，可直接放入 .claude/skills、.codex/skills、AGENTS.md 或 Cursor 规则中，让每个新会话开箱即懂你。项目由单文件 Python 标准库实现，零依赖、无网络请求。
以'挖掘而非配置'的思路切入 AI Agent 个性化，把用户手中每天在流失的会话日志转化为可复用的行为画像，并通过多 Agent 共识机制过滤主观偏差，理念新颖且实现极简。

Fable Mode 工作纪律协议 ⭐ 64

让弱模型跑出前沿模型的工程纪律

Claude Code SkillPython 3 标准库Bash 安装脚本Markdown 规约文档Claude Code Hooks (SessionStart/PreToolUse/PostToolUse/Stop)
fable-mode 是一个面向 Claude Code 的技能包（Skill）加一组守护钩子（Guard Hooks），核心理念是‘输出质量 = 模型能力 × 工作纪律’。它将 Fable 5 级前沿模型的工作习惯抽象成可执行的流程规范，让 Opus 4.8 或任何非前沿模型也能按同等纪律运转。项目通过六大杠杆约束模型行为：Plan Gate（先写 SPEC 再动手编码）、小卡片执行、对抗式自检、真实产品验证、上下文清洁（用外部 SPEC/PROGRESS 替代臃肿会话）以及检查点自治。四个钩子把最容易被模型偷懒的规则转成硬阻断——Profile Injector 会话启动注入规范、Spawn Guard 阻止无计划或越级派发子代理、Fail-Streak Reminder 连续失败时提示归因阶梯、Close Guard 未完成清单不得结束轮次并强制留下证据。整体基于 Python 标准库实现，支持项目级 .fable/ 目录 opt-in，钩子失败开放（fail-open）且循环安全，可安全全局注册。
它把‘Prompt 层的软约束’真正落成‘Hook 层的硬约束’，是少见的从工程流程角度补齐弱模型短板的开源实践，为多模型协同与低成本代理编排提供了可复用的纪律模板。
