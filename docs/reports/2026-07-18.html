BaoCut Agent Skill 是一个开源的 Agent Skill 项目，作为 macOS 视频处理应用 BaoCut 的智能代理封装层，让 Claude Code、Codex 等兼容 skills.sh 规范的 AI 编码代理能够通过自然语言驱动 BaoCut 的命令行工具。项目定位在 AI Agent 与桌面应用之间的桥接层，用户只需用中英文对话（例如「转写并翻译字幕」或输入 /baocut 斜杠命令），代理便会自动调用 baocut CLI 完成视频转写、字幕生成与翻译、说话人识别、口播视频清理及导出等一整套工作流。项目本身不包含转写引擎，而是通过 skills/baocut/bin/baocut 解析器指向 /Applications/BaoCut.app/Contents/MacOS/baocut-cli 完成实际调用。安装方式提供两种：推荐使用 npx skills add 一键安装到 ~/.agents/skills/baocut 并自动关联多个代理，也支持通过 git clone 与符号链接的手动方式部署，无需 Node.js。目录结构包含 SKILL.md（代理入口路由）、references（分任务指南）与 bin（CLI 解析器），遵循 skills.sh 的技能包规范，采用 MIT 协议开源。
该项目是 AI Agent Skill 生态与本地桌面工具结合的典型样板：它示范了如何用极简的目录结构（SKILL.md + references + bin resolver）把一个复杂的 macOS 应用能力开放给多种 AI 编码代理复用，是研究 skills.sh 规范、Claude Code 扩展机制以及 AI 驱动多媒体工作流的优质参考。对内容创作者而言，它把视频剪辑与字幕翻译这类高频操作降维成一句自然语言指令，展示了 Agent 时代垂直工具的新交互范式。

LLMVault 是一个刻意设计为存在漏洞的 CTF 式训练平台，对标 WebGoat/KubeGoat，但专注于 AI 安全领域，聚焦 OWASP LLM Top 10（2025）十大风险类别。项目提供 25 个渐进解锁的实验：核心层 10 个覆盖提示注入、敏感信息泄露、供应链、数据投毒、输出处理、过度代理、系统提示泄露、向量与嵌入、错误信息、无限消耗等类别，每类一题；进阶层 10 个为多轮对话挑战，涵盖角色扮演越狱、碎片重建、Agent 工具链滥用、跨租户 RAG 内存泄漏、基于查询的模型抽取等真实攻击手法；专家层 5 个模拟真实世界已披露的漏洞攻击类，采用 Fernet AES 加密封装在 challenges/expert.enc，需运营方手动发放解密密钥才能解锁。每个实验都配套私有防御指南，形成攻防对照学习。平台采用 Flask 构建，后端脚本化确定性响应，无需 LLM API 密钥即可离线运行，进度以 data/progress.json 文件持久化，支持 Docker Compose 一键部署（gunicorn 单 worker 保证内存计分板一致）。前端提供响应式 UI、悬浮卡片、通关证书 SVG/PNG 导出、社交分享等玩家体验设计。
在 AI 安全教育资源仍稀缺的当下，LLMVault 填补了 WebGoat 类靶场在 LLM 领域的空白：不仅按 OWASP 官方 2025 分类系统组织内容，还创新性地把攻击、防御、多轮交互和真实攻击类合成为分级课程；脚本化确定性响应避免了对付费 API 的依赖，可完全离线复现，非常适合企业培训、CTF 出题人和红蓝队自建演练环境。

deja-vu ⭐ 336

编码 Agent 的本地记忆检索层

GoMCP (Model Context Protocol)SQLite / sqlite3 CLIJSONL 追加式存储Shell 安装脚本npm / Homebrew / Go install 多渠道分发
deja-vu 是一个面向编码类 AI Agent 的本地记忆层工具，将 Claude Code、Codex、opencode、Cursor、Gemini CLI、aider、Antigravity、Grok Build、Qwen Code 等主流 Agent 已经写在本地的会话日志转化为可检索、可召回、可共享的知识库。项目以单一零依赖二进制形式分发，无需下载模型或运行后台服务，所有数据默认留在本地。核心能力包括：命令行全文搜索（deja ，支持多词 AND、正则、按 harness/project/时间/角色过滤，官方声称在 GB 级日志上约 12ms 返回），追溯性索引旧日志；通过 MCP 协议向 Agent 暴露 recall 工具，让 Agent 在对话中主动查询过往解决方案；--auto 模式在 SessionStart 钩子中自动注入相关上下文；索引期即对 API Key、JWT、私钥等敏感信息做脱敏，缓存本身可安全保留；提供 stats 使用统计与 SVG 分享卡、share 生成脱敏会话摘要、sync export/import 与 ssh 双向同步（追加式、幂等、带水印）；doctor 输出机器可读的自检 JSON；resume 直接跳回原生 Agent 会话。安装方式覆盖 curl、go install、npm、Homebrew，deja install --all/--auto 自动写入各 Agent 配置并保留 .bak。整体定位是把散落在各家 Agent 目录下的历史对话真正变成可用的团队与个人记忆。
编码 Agent 生态高速膨胀，但每家工具的会话日志相互隔离、无法搜索，用户每天都在重复解决自己已经解决过的问题。deja-vu 用一个零依赖二进制统一了九款主流 Agent 的历史，把 MCP 作为 Agent 侧接入协议、把本地文件作为可信数据源，既避免了云端记忆产品的隐私顾虑，又通过 SessionStart 自动召回真正做到无感集成，是 AI Agent 时代少见的把工程细节（脱敏、幂等同步、doctor 自检、跨机器 SSH）打磨到位的记忆基础设施。

Claude Lane Stack ⭐ 125

Claude Code (Anthropic)GrokCodex (GPT-5.6 Sol/Terra/Luna)Bash / Shell 脚本GitYAML 任务卡systemd (user-mode)Mermaid
Claude Lane Stack 是一套面向单人开发者的多智能体（multi-agent）编码编排框架，基于 Claude Code 构建，把常见的『五个聊天窗口、手动合并分支』的混乱工作流改造成一条可检查的『AI 流水线』。用户只与一个名为 dev-orchestrator 的项目经理智能体（PM）对话，PM 负责拆分任务、生成 YAML 任务卡（存放于 .agents/runs/），并把工作分发给可选的工人车道：Grok 负责重写入、Codex 负责代码评审与项目 onboard。每张任务卡都声明『拥有路径』（file contracts），确保工人不越界；Codex 组成的评审车道对每次合并把关，检查通过后由 PM 自动合并到 main 分支。项目当前版本 v1.3.1，核心特性包括：Onboard 2.0（minimal/full 场景 + fast/deep 深度双轴，自动为成熟仓库生成『取证式护照』）、lane-ctl 与 user-systemd 结合的后台车道生命周期管理（解决长任务 2 分钟超时问题）、事件驱动的渐进式 accept（并行任务无需等待最慢者）、以及 /resume-project 冷启动恢复命令。整套系统仅依赖纯文本文件加 git，无需任务数据库或云服务，缺失的工人由 agents-doctor 自动降级到 claude-only 模式。文档覆盖 9 种语言，附完整新手指南。
它把『AI 结对编程』升级为『AI 团队协作』，用最朴素的 git 加文本文件方案解决了多智能体协同的三大痛点：上下文重复解释、文件互相覆盖、无人评审。对独立开发者和 vibe-coder 而言，这是少见的兼顾工程严谨性和低门槛的多智能体框架，且完全在本地可见可审计，没有黑盒平台锁定。

AgentStack 是一个基于 Mastra 框架构建的生产级 AI 智能体平台，专注于金融情报、RAG 检索管线、企业级可观测性与安全治理。项目内置 25+ 专业化智能体、57 个企业工具、10+ 工作流、12+ 主管网络（Supervisor Networks）以及 105 个 UI 组件（含 50+ AI Elements 与 55+ shadcn/ui 基础组件），通过 A2A/MCP 协议实现可扩展的智能体协同编排。在金融领域，平台集成了 Polygon、Finnhub、AlphaVantage 等 30+ 数据端点，覆盖行情、基本面、技术指标等场景。RAG 能力方面采用 LibSQL HNSW 向量检索（3072 维嵌入）配合 MDocument 分块、重排序与 GraphRAG 图遍历，实现开箱即用的语义检索。工作区管理支持 AgentFS、Daytona 与本地沙箱三种模式，并通过 LibSQL 持久化智能体、工作区、主管网络与鉴权状态。安全治理层集成 Better Auth、RBAC、路径穿越防护与 HTML 消毒，可观测性通过 TanStack Query 与 Langfuse 追踪每一次智能体调用、工具执行与工作流步骤。技术栈以 Next.js 16、React 19、TypeScript 5、Zod 全量类型校验为核心，测试覆盖率达 97%，主流 LLM 支持 Gemini 2.5、OpenAI、Claude。
