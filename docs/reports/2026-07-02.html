Agent Skills 开放标准Markdown（SKILL.md / AGENTS.md / .mdc）Claude Code SkillsCursor Rulesnpx skills CLIClaude Code 插件市场（marketplace.json）
self-learning-skills 是一个面向 AI 编码代理（Claude Code、Cursor、Codex、Zed、Aider、Gemini CLI 等）的自我改进型元技能。它并不直接执行开发任务，而是教会代理识别一次会话中经过反复调试才走通的『黄金路径』——例如如何连上生产数据库、凭据存放位置、部署命令、线上验证方式——并把这类难得的经验持久化到下次会话可自动加载的位置，避免每个新会话都从零重新摸索。核心工作流分三步：识别时机（多次尝试才成功、非显而易见的命令、易复现的操作流程或用户明示『记住这个』）、无需追问即时捕获（自选作用域与名称，记录过程而非一次性答案，并附带『什么行不通』），以及下次会话自动复用。项目按工具差异化落盘：Claude Code 生成 skills//SKILL.md，Cursor 生成 .cursor/rules/learned/.mdc，其他代理则追加到 AGENTS.md。它还内置分诊机制（多步流程升为技能、单条事实入 MEMORY.md、一次性内容丢弃）和三条晋级规则（必须有通过校验、有命名的失败模式、有排除的死胡同）以避免把猜测固化。安全上明确只记录密钥位置而不写入密钥值。基于开放的 Agent Skills 标准构建，可通过 npx skills、Claude Code 插件市场或手动方式安装。
在 AI 编码代理普遍缺乏跨会话记忆的现状下，本项目提出了一个轻量、可移植且工具无关的元技能范式：不做具体工作，而是捕捉『工作是如何完成的』。其分诊 + 晋级双闸门的设计尤为精巧——既避免了配置膨胀，又通过『必须验证 + 必须记录失败』把知识沉淀从主观感受推向可信证据，还明确禁止写入密钥值以保证团队共享安全。这是一种可以直接迁移到自家代理工作流的方法论级实践。

reverse-flow-skill 是一个面向 AI Agent（尤其是 OpenAI Codex）的本地 CTF 逆向工程流程技能包。项目通过启动词『真心为你』让智能体进入逆向模式，默认假定用户处于本地沙盒、CTF、crackme、wargame 或授权训练靶场环境，无需在每轮对话中重复声明合法性上下文。技能内部采用『英文内核提示词 + 中文用户交互』的双语设计，用英文规则驱动模型稳定性，用中文完成启动、报告和下一步菜单，兼顾模型效果与中文用户体验。核心工作流被拆分成六个稳定阶段：分析、报告、逆向、深度逆向、漏洞研判以及用户选择下一步，使 Agent 能够按照标准化流程完成样本分诊、工具链自检、静态与深度逆向、漏洞研判和报告输出。项目还内置了口语化需求归一化机制，将『解锁 XX』『去除 XX』『绕过反调试』『让它通过』『拿 flag』等模糊表达映射到具体的逆向动作，如定位 feature gate、恢复校验逻辑、生成 patch 或推导期望输入。仓库以 SKILL.md、agents/openai.yaml 以及 scripts/ 下的 create_case、triage_artifact、tool_audit、report_from_triage 等 Python 脚本组织成一个可复制到 ~/.codex/skills 的技能目录，便于快速接入 Codex。整体形态更像是一个『提示词工程 + 流程编排 + 辅助脚本』的组合技能，而非重量级逆向框架。
在 AI Agent 技能生态中，多数项目聚焦通用编码或搜索，而 reverse-flow-skill 把 CTF 逆向这一垂直场景做成了可复用的流程技能，并显式约束在本地沙盒、CTF、crackme 等合法上下文中运行，兼顾合规边界与实战效率。它示范了如何用『英文提示词内核 + 中文交互 + 少量脚本』的轻量组合，把一个专业领域的操作范式沉淀成 Codex 可直接加载的技能包，对想要构建垂直 Agent Skill 的开发者具有直接参考价值。
