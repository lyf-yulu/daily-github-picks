BaoCut Agent Skill ⭐ 132

让AI代理驱动BaoCut视频剪辑

skills.sh Agent Skill 规范Shell / CLI 封装Markdown (SKILL.md)Node.js (npx 安装器)macOS / BaoCut.app
BaoCut Agent Skill 是一个开源的 Agent Skill 项目，旨在让 AI 编程代理（如 Claude Code、Codex 及任何兼容 skills.sh 的代理）通过自然语言驱动 macOS 视频处理应用 BaoCut。它对 BaoCut 应用内置的 `baocut` 命令行工具进行了封装，让用户可以用一句话完成音视频转写、字幕生成与翻译、说话人识别校对、口播视频剪辑以及最终导出等复杂工作流。项目本身不包含转写或翻译引擎，而是作为中间层的技能路由，把用户的自然语言指令翻译成对本地 CLI 的具体调用，例如 `baocut --json auto talk.mp4 --lang zh` 进行一键转写、润色与翻译，或 `baocut export --srt --translated --lang zh` 导出翻译后的 SRT 字幕。安装方式支持通过 `npx skills add` 一键完成，也支持无 Node.js 环境的手动 git clone 加软链方案，将 skill 全局挂载到 `~/.agents/skills/baocut` 并链接到各代理的 skills 目录。项目结构清晰，包含作为代理入口路由的 SKILL.md、按任务划分的 references 指南目录，以及一个能自动定位 `/Applications/BaoCut.app` 内 CLI 的 bin 解析脚本。整个方案仅支持 macOS，采用 MIT 协议开源。
它示范了 Agent Skill 这种新兴形态如何优雅地把桌面原生应用接入 AI 编程代理生态：不重造引擎，只做自然语言到 CLI 的路由与规范，就能让 Claude Code、Codex 直接完成转写、翻译、剪辑等重活。对于想给自家 macOS 工具加 AI 入口的开发者，是一份可直接参考的最小可行样板。

海鸥 codeX 版是一套面向 OpenAI Codex 命令行工具的本地增强配置包，旨在通过自定义配置将通用 AI 编程助手改造为更贴合中文开发者与安全研究人员使用习惯的专业工作台。项目并不重写 Codex 本身，而是围绕 Codex 的扩展点进行分层定制：通过顶层的 AGENTS.md 注入更直接、更符合中文技术表达习惯的人格设定与响应规范；通过模块化的 Prompt 组合，将常用的开发、审计、研究场景拆分为可复用的提示词片段；通过专业 Skill 模块，为渗透测试、漏洞分析、逆向研究等安全场景提供工作流封装。整体思路是把 Codex 视为可编程的 Agent 运行时，将人格、流程、领域知识以配置文件的形式分离出来，便于版本化维护和按需加载。适用人群主要是使用 Codex 进行日常编码、代码审计以及安全研究的中文技术人员，可以在保持官方 Codex 更新的同时，获得更聚焦、更本土化的交互体验，避免每次开新会话都要重复给出上下文与角色设定。
该项目切中了当前 AI 编程工具本地化与专业化的两个关键痛点：一是官方 Codex 默认人格偏英文与保守，中文技术场景下沟通成本较高；二是安全研究这类高度流程化的领域缺乏成体系的 Prompt 与 Skill 封装。作者通过 AGENTS.md、Prompt、Skill 三层结构给出了一个可复制的定制范式，对希望把 Codex 打造成个人化工作台的开发者与红队人员具有直接的参考价值。

MeetingCopilot 是一款面向 Windows 与 Apple Silicon macOS 的实时会议/面试副驾工具，主打「实时听取对方语音 + 第一人称提词回答 + 防录屏捕获」三大能力。Windows 端通过系统 loopback 直接捕获对端音频，无需引入会议机器人；macOS 借助 BlackHole 等虚拟音频设备完成路由，同时独立麦克风通道转写用户自己的发言。ASR 支持四种可切换后端：默认本地 FunASR 流式（Python sidecar 由应用自动拉起与回收）、本地 Whisper turbo（DirectML GPU 离线兜底）、阿里云 fun-asr-realtime 字级流式、以及 MiMo 分段式，并原生支持中英文自动切换与灰色流式部分字幕。回答侧采用 BYOK 的 OpenAI 兼容 LLM（推荐 DeepSeek），生成结论先行、STAR/技术拆解结构化的口播式答案，且严格基于本地解析的简历与 JD，不臆造经历。项目还内置滚动式面试备忘、prefix cache 预热（首答走 KV 缓存命中）、区域截图问 VLM、深浅色主题、中英双语 UI、多会话隔离等工程细节，Windows 端窗口对录屏隐身，macOS 为尽力而为。
把「实时字幕 + 隐身窗口 + 简历接地的提词回答」这条链路做到了工程可用：本地 Python ASR sidecar 由 Electron 自动托管、LLM 侧用 1-token 请求预热 prefix cache 压低首答延迟、Windows 走系统 loopback 免装虚拟声卡、并配合窗口反录屏 API，是同类「面试外挂」中少见地把延迟、隐私（本地 ASR + 本地简历解析）和防截屏都认真处理过的开源实现。

deja-vu（deja） ⭐ 302

编码 Agent 的本地记忆检索层

Go（cmd/deja 主二进制）MCP（Model Context Protocol，stdio server）SQLite（读取 opencode / Cursor 索引所用的 sqlite3 CLI）JSONL（同步与会话日志格式）Shell / SSH / SCP（跨机同步通道）npm 包 @vshulcz/deja-vu、Homebrew tap 与 GitHub Releases 分发
deja-vu 是一个面向编码 Agent 的本地记忆层，把 Claude Code、Codex、opencode、aider、Gemini CLI、Cursor、Antigravity、Grok Build 等工具默认写入本地的会话日志聚合成可检索的历史知识库。它以一个零依赖的单文件二进制形式分发，无需下载模型或运行后台服务，全部数据默认留在本机。核心能力围绕检索与召回展开：命令行 deja 可在数 GB 日志上做毫秒级的多词与子串匹配搜索，deja ctx 输出可直接拼进 Prompt 的 Markdown 摘要；通过 deja install 一键把 MCP recall 工具接入各 Agent，让模型主动询问 “这个问题我们上次是怎么解决的”，--auto 模式则通过 SessionStart 钩子在会话启动时自动注入相关记忆。工程细节上，索引时会剥离 AWS Key、JWT、私钥等敏感信息，share 生成的会话摘要天然可安全外发；sync export/import 与 sync ssh 提供 append-only、幂等的多机同步；stats、statusline、resume、sources 等命令覆盖统计、状态栏、原生 Harness 中恢复会话、数据源检视等运维场景。安装渠道涵盖 curl 脚本、go install、npm 与 Homebrew，并支持 deja update 自校验升级。
编码 Agent 生态里“记忆”一直是缺口：每个 Harness 都在本地生成大量对话日志，却各自为政、无法检索。deja-vu 用一个零依赖二进制解决了跨 Harness 索引、MCP 召回、自动上下文注入、密钥脱敏与多机同步这一整套问题，工程完整度高（含 CI、OpenSSF Scorecard、校验升级），并且默认离线不外传数据。它把“Agent 记忆”从概念变成可以立刻接入现有工作流的基础设施，对每天用 Claude Code / Codex / Cursor 的开发者有直接生产力价值。

Marketing Studio 是一个面向 Claude Code 的 Agent 驱动型营销素材生成引擎。开发者只需在自己的产品仓库中输入 `/marketing` 命令，Agent 便会自动完成品牌调研、应用录制、素材渲染并将成品回传，一次运行即可产出六类完整发布物料：Blender + Remotion 制作的 Logo 揭示动画、带镜头推拉与光标的 Playwright 产品演示、30 至 90 秒的发布视频、ElevenLabs 生成的配音与配乐、按 X/LinkedIn/TikTok 分平台的社交短片，以及 OG 图、动态 OG 循环和 README GIF。流水线执行顺序经过精心设计：先跑成本最低的合成以尽早暴露品牌 token 问题，产品演示只录一次并复用给所有下游素材，音频在画面锁定后才配乐；磁盘上的 manifest 支持中断续跑。项目还内置内容简报生成、文案 Linter（拦截破折号、炒作词与 AI 味用语）、按品牌调节的胶片调色与运动风格、多宽高比响应式导出、名为 Mission Control 的本地审批画廊、成套音效库以及可直接粘贴的社媒发布包。品牌数据以 `brands/.json` 配置并经 zod 校验，模板不硬编码品牌值，新增产品只需一份 JSON 与一个 Logo 组件。所有渲染集中在本仓库完成，产品仓库保持干净。
它把 AI Agent 从写代码延伸到了产品营销全链路交付，用一条命令替代了独立发行者原本需要设计师、剪辑师、配音师协作数周的工作；工程上也很讲究：先便宜后昂贵的渲染排序、content-hash 缓存避免重复录屏、断点续跑的 manifest、以及 zod 校验的品牌 token 体系，展现了把 Agent 编排做成生产级流水线的成熟思路，对希望复用 Claude Code Skills 构建垂直自动化的开发者极具借鉴价值。
